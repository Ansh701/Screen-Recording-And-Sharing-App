## 🏁 Getting Started

### **Prerequisites**
- [Node.js v18+](https://nodejs.org/en/)
- [pnpm](https://pnpm.io/) (or npm/yarn)
- [Git](https://git-scm.com/)

### **Installation**
```bash
git clone https://github.com/your-username/capto.git
cd capto
pnpm install
pnpm db:push
pnpm dev
````
